# Требования к приложению Student Task Manager

## 1. Функциональные требования
- Добавление новой задачи
- Удаление задачи
- Редактирование задачи
- Установка дедлайна
- Отметка задачи как выполненной
- Просмотр списка задач
- Фильтрация задач по категориям

## 2. Нефункциональные требования
- Простой и понятный интерфейс
- Быстрая работа приложения
- Работа без подключения к интернету
- Хранение данных локально (на компьютере пользователя)

## 3. Категории задач
- Учёба
- Личное
- Работа
